## Выпуск сертификата

Создание приватного ключа (rsa), создание запроса на подпись и подпись самим собой.

```bash
openssl req -newkey rsa:2048 -nodes -keyout shop.key -x509 -days 365 -out shop.crt 
```

Главное указать CN

![CN](../misc/images/gen_certs.png)
